CSS3 Snowflakes

This is a simple demo of CSS3-based snowflakes. It is based on the work of Estelle Weyl (@estellevw). These snowflakes are dynamically created on page load and several effects (e.g. opacity, size, speed) are randomly added to each snowflake. The code is available on GitHub.

Browser Support

This demo should support Safari, Chrome, and Firefox.

Requirements

If you decide you want to initialize the snowflakes AMD in your <head> tag then you'll need to use a simple library to tell you when the DOM is ready like domReady. Please read the documentation on how to use domReady AMD before using it. The domReady AMD library is NOT included in this distribution, but you can download it at Github.
